# 电商经营分析｜1页业务结论（模板）

> 用途：把 Tableau 看板和 SQL 分析的结论浓缩成 1 页，投递/面试展示很加分。
> 建议：填完后导出为 PDF（命名：`Olist_Analysis_Summary.pdf`）。

---

## 1. 背景与目标
- 数据来源：Olist Brazilian E-Commerce Public Dataset（公开业务数据）
- 业务目标：构建电商经营监控体系，识别影响 GMV/体验的关键因素，并提出可执行优化建议。

---

## 2. 指标口径
- 有效订单：仅统计 `order_status = 'delivered'`
- 主口径：`item_gmv = sum(price + freight)`（订单商品明细汇总）
- 对账口径：`paid_gmv = sum(payment_value)`（支付流水汇总）
- 差异说明：见 `docs/metric_dict.md`

---

## 3. 核心结论（填 3-5 条）
1) 【趋势】订单与 GMV 在 2017 年持续爬升并在 2017-11 达到峰值（订单 7,289；paid_gmv≈1,153,528），旺季后（2018-02~2018-03）履约承压，准时率降至 0.845/0.789 且差评率升至 0.185/0.207。

2) 【结构】GMV 与订单高度集中在核心州：SP（订单 33,493；GMV≈4,749,986）显著领先，其次为 RJ（≈1,641,184）、MG（≈1,493,364）。高客单类目（如 watches_gifts，AOV≈221.6）对 GMV 拉动明显。

3) 【体验】整体漏斗健康：下单 99,441 → 支付完成 98.76% → 已送达 97.02% → 已评价 95.91%。差评与延迟强正相关：延迟 0 天差评率 9.10%，延迟 1-3 天 31.50%，延迟 4-7 天 65.15%，延迟 8+ 天 76.62%；交付 22+ 天差评率达 40.33% 且准时率仅 45.79%。

4) 【用户】整体复购偏低：总用户数 93,358，复购用户数 1,693，复购率 1.81%；M1 留存 0.451%。RFM 分层中 Champions（984，≈1.05%）与 Loyal（1,456，≈1.56%）规模小但价值高（Champions 平均频次 2.18、客单/贡献更高）；At Risk 20,835（≈22.3%）存在可挽回空间。

---

## 4. 建议（写 2-3 条可执行）
- 建议 1：优先治理“延迟订单”（delay_days>0），将延迟 1-3 天订单差评率从 31.5% 拉回至不延迟水平（9.1%）；旺季（2017-11~2018-03）提前扩容干线/末端与仓配产能，降低 22+ 天交付长尾。
- 建议 2：按区域设定分层 SLA 与补偿策略，优先覆盖差评率高且交付慢的州（如 RJ/BA/CE/MA）；对高价值用户（Champions/Loyal）提供延迟预警与补偿券，降低差评并提升复购。
- 建议 3：建立监控看板与告警：准时率、延迟分布（1-3/4-7/8+）、差评率按州/类目/卖家拆解，形成“发现-定位-复盘-优化”闭环。
