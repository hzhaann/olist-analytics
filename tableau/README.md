# Tableau 交付物（仅截图）

本项目的可视化交付物采用“只提交截图”的方式（使用 Tableau 制作），避免工作簿文件体积过大或包含本地连接信息。

## 截图存放位置

将截图放到：

- `tableau/dashboard_screenshots/`

## 截图命名规范（建议）

- `tableau_p1_overview.png`
- `tableau_p2_growth_drivers.png`
- `tableau_p3_funnel_retention.png`
- `tableau_p4_delivery_reviews.png`
- `tableau_p5_rfm.png`

## 导出/截图建议

- 在 Tableau 中切换到对应页面
- 选择 16:9 画布（如 1920x1080）
- 使用系统截图工具保存为 PNG
- 确保截图中包含：
  - 页面标题
  - 时间筛选器（如有）
  - 关键 KPI 卡片与核心图表

在仓库 `README.md` 或 `reports/analysis_report.md` 中引用截图示例：
